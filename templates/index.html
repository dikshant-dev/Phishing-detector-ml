About PhishDetector

Phishing is an internet scam in which an attacker sends out fake messages that look to come from a trusted source. A URL or file will be included in the mail, which when clicked will steal personal information or infect a computer with a virus. Phish Detector is a project which aims to help reduce phishing attack by helping internet users authenticate URL links.
